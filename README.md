# Kanban - React

Repositório destinado a projeto de Kanban dinâmico e responsivo, usando React.

- Adicionar tarefas na lista
- Atualizar tarefas já adicionadas
- Deletar tarefas indesejadas
- Mover tarefas entre listas

## Feito com
- HTML
- CSS
- JavaScript
- React
